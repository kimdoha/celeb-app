# celeb-app
SNS 속 셀럽들이 픽한 핫 플레이스들을 한 눈에 볼 수 있는 플랫폼입니다. 

유튜브나 인스타그램 속 셀럽들의 핫 플레이스 장소들을 모아 소개합니다.
